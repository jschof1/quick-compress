# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-02-13

### Added

- Initial release of Quick Compress
- WebP conversion with automatic optimization
- Smart resizing (max 1280x1280)
- Intelligent quality and resize adjustment to target files under 200KB
- Support for JPG, PNG, GIF, BMP, and TIFF formats
- Batch processing for entire folders
- macOS Finder Quick Actions integration
- One-command installation script
- Comprehensive documentation and README

### Features

- Binary search algorithm for optimal quality vs file size
- Automatic ImageMagick version detection (v6/v7)
- Native macOS notifications
- No dependencies except ImageMagick
- Single file or batch processing
